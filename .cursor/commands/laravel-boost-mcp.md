# laravel-boost MCPを確認

## 概要

ユーザーの入力内容に対してlaravel-boost MCPに確認をしてから返答を行う。実装の際も指定があれば必ずlaravel-boost MCPで一度確認を行う実装する。

## 前提条件

- ユーザーから質問があった
- ユーザーから実装の指示があった

## 実行手順

1. ユーザーの入力内容を確認する
2. 入力内容に対する返答や実装をlaravel-boost MCPで確認する(Search-Docsは日本語対応していないので使う場合は入力内容を英語にして使う)
3. MCPの内容を加味し返答や実装を行う

## ノート

- 返答や実装の原則は、`.cursor/rules/main.mdc` のルールに従ってください。
- プロジェクトの内容を必ず確認して実態に合わせた返答、実装を行ってください。
